code-complexity-analyzer v1.0.0

Released: July 4, 2025

Description: Code Complexity Analyzer | A one-stop, pure-Python analyzer that estimates algorithmic complexity (Big-O), calculates cyclomatic complexity, Halstead metrics, maintainability index, and flags basic code smells & security issues. Perfect for CI gates, refactoring insight, and teaching.

Key Features

  • Heuristic Big-O estimation per function with detailed analysis
  • Cyclomatic complexity and comprehensive branching analysis
  • Halstead metrics & maintainability index calculations
  • Code-smell detection & security pattern identification
  • Flexible CLI & programmatic API for seamless integration
  • Zero dependencies - lightweight and fast

View Documentation | Learn More

Installation

pip install code-complexity-analyzer

Quick Usage

import code_analyzer as ca

# Analyze a single file
ca.analyze_file("example.py")

# Analyze entire directory
ca.analyze_directory("src/")

# Generate detailed reports
ca.generate_report("src/", output="complexity_report.json")

CLI Usage

# Quick analysis
complexity-analyzer src/

# Detailed report with thresholds
complexity-analyzer src/ --format json --output report.json --threshold 10
